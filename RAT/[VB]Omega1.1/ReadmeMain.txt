       A Remote Administration Tool for the Windows 98/98SE/ME 
	             Omega v1.10 written by TSH
                           Copyright © 2001

  This software is a remote administration tool for use with windows,
  Omega is a Remote Administration Tool like Sub7 and it is developed
  by Microsoft Visual Basic 5.0. 

Omega Feature:

Address Book
  -Add, delete, modify IP address, port number, comments.
  -Ping selected IP address
  -Connect selected IP address   
  -Ping All IP address
 
IP Tool
  -Resolve host name is change hostname to IP address.
  -Reverse is change IP address to hostname.
  -Ping is used to make sure that the user is online now.
  -Clear is to clear the result in the list.

Connection
----------
1)IP Scanner
  -This tool scans a range of IP address for victims.

2)Get PC Info
  -Get victim personal computer information

3)Passwords
  -Get cached passwords

4)Server Option
  -Change the port
  -Default port is 54896
  -Set password
  -Remove password
  -Enable key logger
  -Disable key logger
  -Change startup method: win.ini, registry run, or run services
  -Disconnect the victim from Internet
  -Restart the server
  -Remove the server
  -Close server

5)IP Notify
  -ICQ pager notify
  -Email notify.

Advanced
--------
1)Keyboard
  -Open key logger
  -Send Keystrokes
  -Get offline keys
  -Clear offline keys
  -Disable keyboard

2)Chat
  -Chat with the victim

3)Browser
  -Open Web Browser to specified location. 

4)Find Files
  -Search file
	
5)Local Option
  -Set the local folder for downloaded files

  -Maximize the selected window.

  -Disable X button.

4)Process Manager
  -Refresh

5)Clipboard Manager
  -Read clipboard text
